# git-chipper

An interactive CLI for batch deleting local branches.


## Install

```
$ npm install --global git-chipper
```


## CLI

```
$ npm install --global git-chipper
```

```
$ git-chipper --help

  Usage
    $ git-chipper
    # ...follow the terminal prompt instructions
```
